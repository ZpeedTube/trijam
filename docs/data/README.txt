#Data Files

Just a little info about the data file.
Comma ',' is used to separete cells in the csv file.
3 arrows '>>>' is used to make the cell text into a link, with the link after '>>>'.
'&&&' can be used to separete a cell if there is multiple winner in one cell.

Example:
58,Donitz>>>https://donitz.itch.io/,FlipFlip>>>https://donitz.itch.io/flipflip,0 Buttons
